GamsPRI Gamificador de Sprints

HOME

Esta plataforma tem o objetivo de gerenciar e gamificar as sprints realizados pela fábrica de software, utilizando técnicas de jogos em contextos não jogáveis.

O GamsPRI é um sistema de pontuação, responsável pelo controle da execução de cada uma das sprints, o sistema apresentará um ranking de pontuações para determinadas situações durante a execução de uma sprints, bem como as regras que deverão ser obedecidas.

A partir dos core drive: Significado Épico e Chamado, Desenvolvimento e Realização e Empoderamento e Posse, foram selecionadas técnicas a serem utilizadas como base para o desenvolvimento da gamificação das sprints, visando melhorar o desempenho dos envolvidos.

Com base nos core drive selecionados foram elaboradas opções para a visualização e interação do usuário com o seu progreso, o usuário poderá verificar seus progresso, seus trofeus, medalhas e rank. a tabela abaixo apresentaráa legenda de opções.
